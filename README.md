# The Movie Apps

- desain Arsitektur = MVVM
- Depedency Injection = Koin
- Reactive Programming = Coroutin Flow 
- ViewBinding
- Navigation
- Paging2
- Shimmer
- Retrofit 
- ViewPager2
- Glide
